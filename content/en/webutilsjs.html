WebutilsJS

WebutilsJS is a free open source web framework for web user interface utilities.

WebutilsJS framework has common web user interface utilities as Grid, Menu, Tabs, Slideshow.

This project was developed for and can be used with WebJS framework, but it is not dependent on it, can be used also with other projects and technologies.
