AspAuth Software

AspAuth is a website template with Asp.Net webforms authentication.

Websites for complex business processes as payments on line require authentication and user management that has to be developed from start, with this project this requirements are satisfied.

This project is ready to use as website for presentation, or can be used as starting point for complex websites and business processes.
